# Directorio de Imágenes - LDX Software

Este directorio debe contener las siguientes imágenes para el correcto funcionamiento del sitio:

## Imágenes Principales
- logo.png (Logo principal - fondo transparente)
- logo-white.png (Logo blanco para fondos oscuros)
- hero-bg.jpg (Imagen de fondo del hero - 1920x1080px recomendado)
- about-team.jpg (Foto del equipo - 800x600px recomendado)
- og-image.jpg (Imagen para Open Graph - 1200x630px)
- twitter-card.jpg (Imagen para Twitter Cards - 1200x600px)

## Favicon

- favicon-16x16.png

## Notas:
- Todas las imágenes deben estar optimizadas para web
- Formatos recomendados: JPG para fotos, PNG para logos con transparencia
- Tamaños máximos recomendados: 1MB por imagen
- Usar herramientas como TinyPNG para optimización

Para generar favicons, puedes usar: https://favicon.io/
Para optimizar imágenes, puedes usar: https://tinypng.com/
